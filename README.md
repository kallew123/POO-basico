## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar **Programação Orientada a Objetos (POO) em Python**.

O sistema simula uma **conta bancária simples no terminal**, permitindo que o usuário realize operações básicas como depósito, saque e consulta de saldo.

## 🧠 Estrutura do sistema

### Classe `ContaBancaria`

A classe representa uma conta bancária e possui os seguintes atributos:

- `titular` → Nome do dono da conta  
- `saldo` → Valor disponível na conta  

---

### Métodos

#### `depositar(valor)`
Adiciona um valor ao saldo da conta.

#### `sacar(valor)`
Realiza um saque, verificando se há saldo suficiente.

Caso não haja saldo:

Saldo insuficiente!


#### `mostrar_saldo()`
Exibe o saldo atual da conta.

---

## 🖥️ Funcionamento

Ao iniciar o programa, o usuário informa:

- Nome do titular
- Saldo inicial

Em seguida, um menu interativo é exibido:


1 - Depositar
2 - Sacar
3 - Mostrar saldo
4 - Sair


O sistema continua rodando até o usuário escolher a opção de saída.
